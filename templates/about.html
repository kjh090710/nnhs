About

오존 농도 분석 결과 대시보드입니다.

2026년 남녕고등학교 데이터 랩 퀘스트 최종 산출물입니다. colab에서 분석한 결과를 보기 쉽게 정리했습니다.

Project type 데이터 분석 웹
대상 기간 2010~2025
대상 월 6~8월

Workflow

탐구 진행 흐름

데이터 통합부터 예측 평가까지의 과정을 4단계로 정리했습니다.

01

데이터 통합

146개의 분산된 데이터를 2개의 데이터로 통합했습니다.

02

탐색 분석

상관관계, 산점도, 시간대별,월별,연도별 평균 변화를 확인했습니다.

03

모델 비교

선형회귀, Ridge, Polynomial Ridge, SGD ElasticNet, HistGradientBoosting 성능을 비교했습니다.

04

예측 평가

실제값과 예측값 그래프, 잔차 산점도, 잔차 분포를 통해 모델 한계를 확인했습니다.

Structure

웹 구성 요소

각 페이지는 분석 결과를 다른 관점에서 보여주도록 구성했습니다.

홈 전체 분석 흐름과 핵심 지표를 요약합니다.
상관관계 오존과 기상,대기오염 변수 사이의 관계를 확인합니다.
시간대 분석 시간, 월, 연도에 따른 오존 농도 변화를 비교합니다.
모델링 결과 여러 회귀 모델의 R², MAE, RMSE를 비교합니다.
예측 및 평가 입력값을 조정하며 오존 농도 예측 결과를 확인합니다.
